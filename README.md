# Well Log Visualization

A Python-based mini project for visualizing subsurface well log data using the FORCE 2020 Lithology Prediction Dataset.

## Project Overview

This project provides quick visualization tools for common petrophysical logs used in reservoir characterization and formation evaluation:

- Gamma Ray (GR)
- Resistivity Logs (RDEP, RMED, RSHA)
- Bulk Density (RHOB)
- Neutron Porosity (NPHI)

---

## Dataset

Dataset used:

FORCE 2020 Lithology Prediction Dataset

Source:
https://www.kaggle.com/datasets/faresazzam/well-logs-dataset-for-machine-learning
